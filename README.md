# Programadores para la Paz 💻🕊️

Bienvenido a **Programadores para la Paz**, un proyecto enfocado en usar la tecnología como herramienta de transformación social, aprendizaje colaborativo e impacto positivo en la comunidad.

## 🌎 Objetivo del proyecto

Este repositorio busca reunir desarrolladores, estudiantes y personas apasionadas por la tecnología para crear soluciones que promuevan:

* La educación tecnológica
* La inclusión digital
* El trabajo colaborativo
* La innovación social
* La construcción de paz mediante la programación

Creemos que el código también puede generar cambio social.

## 🤝 Cómo contribuir

Las contribuciones son bienvenidas.

1. Haz un fork del proyecto
2. Crea una rama:

```bash
git checkout -b feature/nueva-funcionalidad
```

3. Realiza tus cambios
4. Haz commit:

```bash
git commit -m "Agrega nueva funcionalidad"
```

5. Sube los cambios:

```bash
git push origin feature/nueva-funcionalidad
```

6. Abre un Pull Request
